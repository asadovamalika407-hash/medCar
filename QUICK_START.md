# 🚀 TEZKOR ISHGA TUSHIRISH

## ✅ TAYYOR BO'LGAN:

1. ✅ **Backend API** - Node.js + Express + MongoDB
2. ✅ **HR Dashboard** - Xodimlar boshqaruvi sahifasi
3. ✅ **Database** - MongoDB Atlas (allaqachon ulanadi)

---

## 🎯 ISHGA TUSHIRISH:

### **1-QADAM: Backend serverni ishga tushiring**

Yangi terminal oching:
```bash
cd backend
npm install
npm start
```

**Natija:** Backend ishga tushadi - http://localhost:5000

---

### **2-QADAM: Frontend ochish**

VS Code da `index.html` ni oching va **"Go Live"** tugmasini bosing.

**Natija:** Website ochiladi - http://localhost:5500

---

### **3-QADAM: HR Dashboard ga kirish**

Brauzerda:
```
http://localhost:5500/pages/hr-dashboard.html
```

## 📊 API ENDPOINTS:

- `GET /api/employees` - Xodimlar ro'yxati
- `POST /api/employees` - Yangi xodim
- `PUT /api/employees/:id` - Xodimni yangilash
- `DELETE /api/employees/:id` - Xodimni o'chirish
- `GET /api/attendance` - Davomat
- `POST /api/auth/login` - Login
