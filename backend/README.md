# MedCare Backend API

## 🚀 O'RNATISH:

### 1. Paketlarni o'rnatish:
```bash
cd backend
npm install
```

### 2. Serverni ishga tushirish:
```bash
npm start
```

## 🌐 API Endpoints:

- **Auth:** http://localhost:5000/api/auth
- **Employees:** http://localhost:5000/api/employees
- **Attendance:** http://localhost:5000/api/attendance
- **Patients:** http://localhost:5000/api/patients
- **Doctors:** http://localhost:5000/api/doctors
