## 🔧 TERMINALDA BAJARISH KERAK:

### 1. Serverni to'xtatish:
Terminal da **Ctrl+C** tugmalarini bosing (server to'xtaydi)

### 2. Material-UI paketlarni o'rnatish:
```bash
npm install @mui/material @emotion/react @emotion/styled @mui/icons-material
```

### 3. React Router o'rnatish:
```bash
npm install react-router-dom
```

### 4. Axios o'rnatish:
```bash
npm install axios
```

### 5. Date-fns o'rnatish:
```bash
npm install date-fns
```

### 6. Serverni qayta ishga tushirish:
```bash
npm run dev
```
